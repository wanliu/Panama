Welcome to Panama

你现在看到是万流软件公司开发的 B2B 电商平台项目，内部代号 Panama, 本项目代码 及项目代码中的一切资源，包括文档，说明书，图片，声音，多媒体，数据等所有资料 版权均归属万流软件公司所有。

Panama 是面向 B2B 下游链中的社区网站，目标受众群体是终端经销商和与之服务的供 应商，批发商。Panama 所完成的功能，现阶段主是要社区与活动展示与索引分类数据， 并且包括了交易平台，目前也是万流平台主要入口。

入门

启动项目

  1. 在命令控制台下输入

    git clone git@github.com:wanliu/Panama.git Panama

    cd Panama

    rails s

  2. 到浏览器地址栏输入 localhost:3000/ 你会看到我们的项目启动画面

    如果一切准备完成，你会幸运的看到软件的画面，不过这一切都要设置完成

组成

Panama 的完整网站是由几大应用服务层组成，其本身就比较复杂，为了保证在功能 上的共用性能上的弹性与可展性，我们会尽量的让各应用服务层任务单一化，所 以可以遇见的未来软件还会进行进化，会更进一步的化分

Panama 的主要组成由 Panama + elasticsearch + caramal + accounts 组成，Panama 包括了社区，交易系统，实时通信系统等，下列产品分别都是单独服务

  1. Elasticsearch实时搜索系统， 官方网址 www.elasticsearch.org/ 你还要下载一个 elasticsearch plugin ,

  2. CaramalServer 万流内部的实时通信系统，github.com/wanliu/Caramel-Server

  3. Accounts 万流中心帐号服务器，github.com/wanliu/Accounts

定位

Panama 是一个高度实时化的应用，能够满足快速的商业需求，也是一个交互性相当高的 web 应用 提供用户便捷使用操作模式，所以我们选择基于 HTML5 的技术架构上，运行在现代型的浏览器，给 用户提供最佳用户体验
